API Documentation

Welcome to our API. Here is how you can use it:

Endpoints

  • POST /api/users: Create a new user.

POST /api/users

Create a new user by sending a JSON payload with the following structure:

{
    "username": "your_username",
    "password": "your_password"
}

Example:

curl -X POST http://localhost:8080/api/users -H "Content-Type: application/json" -d '{"username": "testuser", "password": "testpassword"}'
